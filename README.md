## Instalação e testes

Primeiro clone o projeto
```
git clone https://github.com/Kallebe18/instagram-copy.git
```
Instale as dependências
```
npm install
```
Inicie o bundler javascript
```
npm run start
```
com o dispositivo ou emulador conectado digite
```
npm num android
```
